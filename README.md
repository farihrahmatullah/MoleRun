# Mole Run

Mole Run adalah game 2D endless runner yang dikembangkan menggunakan Unity dan C#.  
Game ini terinspirasi dari Dino Run, dengan mekanik tambahan berupa kemampuan karakter untuk masuk ke dalam tanah sebagai strategi menghindari rintangan.
